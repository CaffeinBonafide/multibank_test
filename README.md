# MultiBank Trading Platform - Cypress Automation

## Overview

End-to-end tests for https://trade.multibank.io using Cypress + TypeScript.

## Prerequisites
- Node.js 18+
- npm 8+

## Setup
```bash
npm install
npx cypress verify
npm run type-check
```

## Run Tests
```bash
# Open runner
npm run cy:open

# Headless run
npm run cy:run

# Specific browser
npm run cy:run:chrome
